## How to Add New Problems

1. Create a folder: `leetcode/difficulty/problem-number-name/`
2. Add files following the template above
3. Update the progress sections
4. Commit with message: `[LEETCODE] Problem #123: Problem Name - Completed`

## Commit Message Format

```

```

## Tips for Problem Solving

1. **Understand the Problem**
   - Read carefully
   - Identify constraints
   - Work through examples

2. **Plan Your Approach**
   - Brute force first
   - Optimize
   - Consider edge cases

3. **Code Carefully**
   - Write clean code
   - Add comments
   - Test thoroughly

4. **Analyze Complexity**
   - Time complexity
   - Space complexity
   - Can it be optimized?

5. **Review Solutions**
   - Understand different approaches
   - Learn from others' code
   - Practice similar problems
