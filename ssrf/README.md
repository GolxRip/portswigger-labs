# Server-Side Request Forgery (SSRF) Learning Path

**Status:** ⭕ Not Started (0/23)

## Overview

This learning path teaches you about server-side request forgery (SSRF). You'll learn about its impact, common techniques used in attacks, and how to defend against them.

## Topics to Cover

### SSRF Fundamentals
- Understanding SSRF
- Common injection points
- Attack scope

### SSRF Attack Techniques
- Basic SSRF
- SSRF with filtering bypass
- Blind SSRF
- Time-based detection
- Out-of-band detection

### SSRF to RCE
- Exploiting internal services
- Interacting with admin panels
- Accessing cloud metadata

### Defense Mechanisms
- Input validation
- Allowlisting
- Network segmentation
- Disable unused protocols
